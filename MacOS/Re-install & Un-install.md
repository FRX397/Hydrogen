- Open Finder

- Search Roblox

- Add/Move Roblox Application To Applications

- done.

# How To Re-install Hydrogen On MacOS

- Open finder

- Search Roblox

- Right Click On Roblox Application

- Show Package Content

- Open Contents Folder

- Open MacOS folder

- Remove the files bellow

“libhydrogen.dylib”

“Roblox Player Back”

- New Search

- Search Hydrogen

- Remove Hydrogen Application

- Remove Hydrogen folder

- Open terminal

- Paste this in terminal

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/VersatileTeam/hydrogen-macos-installer/ab4738e070dc878df22798b281d6535f9dca22ca/install.sh)"
```

- Press enter
