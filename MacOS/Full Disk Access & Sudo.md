# How to give terminal/Hydrogen Full Disk Access

- Open settings 

- Security & Privacy 

- Privacy

- Full Disk Access

- press the plus icon and add terminal / Hydrogen

- done

-------

# Open Hydrogen as sudo

- Open terminal

- paste this in terminal 

```
sudo /Applications/Hydrogen.app/Contents/MacOS/Hydrogen\ MacOS
```
- press enter

- done.
